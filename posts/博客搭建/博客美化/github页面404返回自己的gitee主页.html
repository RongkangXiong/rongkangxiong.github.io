GitHub页面404返回自己的Gitee主页

发表于2021-12-11|更新于2025-01-28|博客搭建博客美化
|字数总计:311|阅读时长:1分钟|阅读量:

1 Github Pages的404页面自定义

在/myblog/source/下增加一个404.html页面

可以自定义为重定向到自己另外网站的博客页面

2 网页内部资源不存在时的404页面定义

利用404页面和网页重定向功能，让用户访问另一个自己部署博客的网站

在主题文件夹中开启error_404

本地预览时，访问出错的网站是不会跳到404页面的。

如需本地预览，请访问http://localhost:4000/404.html

在根目录/source下面新建文件夹404，在404文件夹下新建404.md

这一步也可以用hexo new page 404完成
